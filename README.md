# Installation
```bash
mv ~/.config/nvim{,.bak}
git clone https://github.com/vincent-teh/nvim-config.git ~/.config/nvim
nvim
```

# Requirements
- [jesseduffield/lazygit](https://github.com/jesseduffield/lazygit.git)
- fd (`sudo apt install fd-find`)
- tmux
